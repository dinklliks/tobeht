# Tobeht Website

## Project Overview
Static marketing website for **Tobeht** — a full-service performance marketing and consulting agency based in Dubai. Clone of opportunityalbania.com's design patterns with entirely original Tobeht copy.

## Stack
- Plain HTML, CSS, JavaScript — no frameworks, no build tools
- English only

## File Structure
```
tobeht/
├── index.html       ← Home page
├── services.html    ← Services page
├── styles.css       ← Shared styles
├── main.js          ← Nav toggle, smooth scroll
└── CLAUDE.md
```

## Pages
- **Home** (`index.html`): Hero, About, Services preview, Why Tobeht, CTA, Footer
- **Services** (`services.html`): Page hero, 4 service detail blocks, CTA, Footer

## Design
- Sticky top nav: logo left, links right, hamburger on mobile
- Colors: background white/`#F5F7FA`, primary dark `#0D1B2A`, accent gold `#E8A030`, text `#1A1A2E`
- Typography: system sans-serif stack
- Style: clean, corporate, generous whitespace — inspired by opportunityalbania.com layout

## Conventions
- No external dependencies (no CDN, no Google Fonts)
- Mobile-first responsive CSS
- Shared nav and footer across pages — keep markup consistent
- No comments in code unless logic is non-obvious
